#  index.js 



A terminal-based Robot implementation written in JavaScript (Node.js).


## Instruction - Keyname and Commands
space --- Start
esc  --- exit
r --- Get Report
p --- Set StartPoint
m --- Move one Step Forward
a --- Rotate To Left
d --- Rotate To Right

## Installation


# install and run via npm or yarn

```
npm install
```

Run the game.
```bash
node index.js
```
